Obsidion Discord Bot

Obsidion is the worlds most advanced Discord bot for Minecraft that is also free and open source. With Obsidion you can get statistics from your favourite Minecraft server , make custom signs and achievments, and you can link your Minecraft username to the bot so you can request your latest info with the "/profile" command (coming in version 4). Obsidion also has a customizable prefix so that it doesn't interfre with any of the other bots on your server.

Asyncpixel


Asyncpixel is an open source asyncronous python wrapper for the hypixel api with 100% of all endpoints. Asyncpixel is currently the only python library with full coverage of the hypixel API. All of Asyncpixel's models are checked and validated by Pydantic so that our data is always in the correct format for you to use. To install Asyncpixel symply run "pip install asyncpixel".

Obsidion Dashboard

Obsidion Dashboard is the one place you can go to to control everything Obsidion. From the Dashboard you can change the bots prefix so it doesn't interfre with any of the other bots on your server. You can update your linked Minecraft account so you can always get up to date information from the "/profile" command. You can also change the Minecraft server that your Discord server is linked to all from one Website. Obsidion Dashboard will be releasing soon.

Mcsrvstats

Mcsrvstats is an open source asyncronous python wrapper for your favourite Minecraft Servers. Mcsrvstats has support for Hive, Manacube, BlocksMC, UniversoCraft, MineSaga, GommeHD, and VeltPvP. All of Mcsrvstats's models are checked and validated by Pydantic so that our data is always in the correct format for you to use. To install Mcsrvstats symply run "pip install mcsrvstats".
